# esports-dataset
The purpose for this dataset is to show how many users (anonymous players) played esports, with all the popular games listed such as CSGO, Valorant, etc. This dataset shows the year they played, the genre of game it is, and which tournament they played in. The players have shared their locations, and as well as how many wins, they’ve accumulated. 

The target audience are those interested in Esports and want to know about NZ’s current players.
